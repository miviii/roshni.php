Building blocks of system 

  The E-Plastic project is a web-based system that utilizes the Model-View-Controller (MVC) architecture. The system is built using PHP as the backend programming language and MySQL as the database management system. The MVC architecture is chosen because it allows for a clear separation of concerns between the data (Model), the presentation (View), and the business logic (Controller).
